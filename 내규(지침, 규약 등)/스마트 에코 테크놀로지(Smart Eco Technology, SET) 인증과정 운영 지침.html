스마트 에코 테크놀로지(Smart Eco Technology, SET) 인증과정 운영 지침

소관부서 : 대학원교학팀 1202
제정 2022.1.13.(한동대학교전자결재-22-00307(2022.1.13))
개정 2023.6.16.(전자결재-대학원교학팀-113(2023.6.16.))
개정 2024.4.9.(전자결재-대학원교학팀-143(2024.4.9.))

| 부 칙 |

제 1 조 (목적)
이 세칙은 경북지역 풍력에너지 인재양성사업단(이하 "사업단") 스마트 에코 테크놀로지 인증과정 운영에 관한 사항을 규정함을 목적으로 한다.

제 2 조 (프로그램)
① 공간설계공학과 및 첨단융합학과 대학원에 스마트 에코 테크놀로지 인증과정 (Smart Eco Technology, SET) 프로그램(이하 "SET 인증프로그램")을 설치한다.(개정 2024.4.9.)
② SET 인증프로그램은 사업단 시작일부터(2021.5월) 사업 종료시까지(2025년.12월) 운영한다.

제 3 조 (프로그램 운영)
① 공간설계공학과 및 첨단융합학과 대학원은 사업단 참여학생 대상으로 SET 인증 프로그램을 운영할 수 있다.(개정 2024.4.9.)
② 이수자에게 대학 총장 명의의 SET 인증프로그램 이수 증명서를 발급하며 프로그램 명칭은 표1과 같이 기재한다.
③ SET 인증 프로그램 세부 운영사항은 SET 운영위원회에서 결정할 수 있다.
④ SET 운영위원회는 사업단에 참여하는 공간설계공학과 교수로 구성하며, 사업단 협력기업 전문가 1인 이상을 포함한다.

제 4 조 (인증대상 및 참여)
① 공간설계공학과 석사과정 및 첨단융합학과 박사과정 학생을 대상으로 참여 신청을 받을 수 있다.(개정 2024.4.9.)
② 매 학기마다 참여 신청을 문서로 받으며 SET 인증과정 운영위원회에서 참여학생을 선발한다.

제 5 조 (인증 이수 조건)
① 인증프로그램에 소속된 학생은 SET 인증과정 운영위원회에서 결정한 인증과정 이수조건을 만족시켜야 하며 세부 내용은 표 2와 같다.
② 다만, SET 운영위원회에서 필요하다고 인정하는 경우에는 예외로 한다.
③ 인증과정 프로그램 이수조건 커리큘럼은 SET 인증과정 운영위원회에서 결정할 수 있다.
